## Brief Descriptions

**Exp 1 — Adder Circuits**
Design and simulation of Half Adder, Full Adder, and 4-bit Ripple Carry Adder using logic gates. Verified binary addition and carry propagation via truth tables.

**Exp 2 — MUX & Encoder**
Implementation of an 8x1 Multiplexer for data selection using 3 select lines, and an 8x3 Encoder for converting 8 active input lines into a 3-bit binary output.

**Exp 3 — 4-bit Ripple Carry Adder**
Extended study of the 4-bit RCA focusing on carry ripple delay. Used Logisim's chronogram to visualize signal propagation across adder stages.

**Exp 4 — GDB Debugging**
Implemented Stack, Queue, and Singly Linked List in C. Used GDB breakpoints to inspect memory addresses, variable values, and runtime behaviour of each structure.

**Exp 5 — Addressing Modes**
Demonstrated all major CPU addressing modes (implicit, immediate, direct, indirect, register, indexed, displacement, relative, stack) through a single annotated C program.

**Exp 6 — Common Bus**
Designed a 4-bit shared bus using MUXes and D flip-flop registers. Demonstrated controlled data transfer between registers using WE and select signals.
